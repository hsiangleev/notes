1.8 数组截取--$slice

数组截取

正数则从前向后，负数从后向前
初始
{
    "_id" : 1.0,
    "arr" : [ 
        1, 
        2, 
        3, 
        4, 
        5
    ]
}

// 截取数组arr后三位
db.getCollection('user').update(
    {"_id": 1.0},
    {$push: {"arr": {$each: [], $slice: -3}}}
)
结果
{
    "_id" : 1.0,
    "arr" : [ 
        3, 
        4, 
        5
    ]
}
